# Cubemap Demo

Small demo. Made just for fun using really outdated stuff.

## Tech Stack

* C99
* OpenGL 2.1
* GLUT

## Setup

Project uses macOS built-in libraries and frameworks and CMake.

## Build

Project uses CMake for build:

```sh
cd build
cmake ..
cmake --build .
```

## Run

From the project root:

```sh
./bin/cubemap_demo
```
